# Screenshots Directory

This directory contains screenshots for the Chrome Web Store listing.

## Required Screenshots

Place your screenshots here with the following naming convention:

```
01-real-time-translation.png    (1280x800 or 640x400)
02-one-click-search.png         (1280x800 or 640x400)
03-multi-engine-support.png     (1280x800 or 640x400)
04-beautiful-design.png         (1280x800 or 640x400)
05-features-overview.png        (1280x800 or 640x400)
```

## Requirements

- **Dimensions:** 1280x800 or 640x400 pixels
- **Format:** PNG or JPEG
- **Count:** 1-5 screenshots (recommended: 5)
- **File Size:** Max 5MB per image

## How to Create

See `../SCREENSHOT_GUIDE.md` for detailed instructions on creating screenshots.

## Promotional Images (Optional)

If creating promotional tiles, place them here:

```
promo-small.png     (440x280)
promo-large.png     (920x680)
promo-marquee.png   (1400x560)
```
