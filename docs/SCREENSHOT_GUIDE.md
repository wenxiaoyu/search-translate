## How to Create Screenshots

### Method 1: Browser Screenshots (Recommended)

1. **Install the extension** in Chrome
2. **Visit a search engine** (e.g., Google.com)
3. **Trigger the translation** by typing Chinese text
4. **Use Chrome DevTools** to set viewport size:
   ```
   Press F12 ‚Üí Toggle device toolbar (Ctrl+Shift+M)
   Set custom size: 1280x800
   ```
5. **Capture screenshot:**
   - Windows: Win+Shift+S
   - Mac: Cmd+Shift+4
   - Or use Chrome's built-in screenshot tool

### Method 2: Design Tool

Use Figma, Photoshop, or Canva to create polished screenshots:

1. **Take raw screenshots** using Method 1
2. **Import into design tool**
3. **Add annotations:**
   - Arrows
   - Text labels
   - Highlights
   - Borders
4. **Export as PNG** at 1280x800

### Method 3: Screenshot Tools

Use dedicated screenshot tools:
- **Snagit** (Windows/Mac) - Professional annotations
- **Lightshot** (Windows/Mac) - Quick and easy
- **Greenshot** (Windows) - Free and open source
- **Skitch** (Mac) - Simple annotations
